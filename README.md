# 📋 Nota Karet Digital Pro
**Sistem Rekam Timbangan & Manajemen Nota Digital**

![Gun Original Project](https://img.shields.io/badge/gun-ORIGINAL_PROJECT-003366?style=for-the-badge&logo=appveyor&logoColor=white&labelColor=00529b)

[![Version](https://img.shields.io/badge/version-1.0.0-emerald)](https://ragilgunawan.my.id)
[![Tech](https://img.shields.io/badge/Tech-HTML5%20%7C%20TailwindCSS%20%7C%20JS-blue)](https://tailwindcss.com)

**Nota Karet Digital Pro** adalah aplikasi berbasis web (PWA-ready) yang dirancang khusus untuk mempermudah pencatatan hasil timbangan karet bagi pengepul atau petani. Aplikasi ini menggantikan pencatatan manual dengan sistem digital yang akurat, lengkap dengan fitur cetak struk ke printer Bluetooth.

---

## ✨ Fitur Utama

* **⚡ Input Berat Multi-Baris:** Memungkinkan input banyak kantong/satuan berat dalam satu nota secara dinamis.
* **📊 Statistik Real-Time:** Dashboard ringkas yang menampilkan total berat untuk periode:
    * Hari Ini
    * Minggu Ini
    * Bulan Ini
* **🧾 Struk Digital & Cetak:** Generate struk otomatis dengan desain ala kasir minimarket. Mendukung pencetakan langsung ke **Printer Thermal Bluetooth (58mm)**.
* **🔍 Pencarian Cepat:** Mencari data petani atau transaksi lama hanya dengan mengetik nama.
* **💾 Penyimpanan Lokal (Local Storage):** Data tersimpan aman di browser tanpa memerlukan database server.
* **📱 Responsive & Modern UI:** Didesain dengan *Plus Jakarta Sans* dan *Tailwind CSS* untuk pengalaman maksimal di HP.

---

## 🛠️ Teknologi yang Digunakan

* **HTML5 & CSS3:** Struktur dan animasi kustom.
* **Tailwind CSS:** Framework styling untuk tampilan yang bersih dan modern.
* **Vanilla JavaScript (ES6+):** Logika kalkulasi dan manajemen data.
* **Local Storage API:** Untuk persistensi data pengguna.
* **Print API:** Optimasi layout khusus untuk kertas thermal 58mm.

---

## 🚀 Cara Penggunaan

1.  **Input Data:** Masukkan nama petani dan harga karet per kilogram.
2.  **Input Berat:** Masukkan berat karet. Gunakan tombol `+` jika ada banyak timbangan.
3.  **Simpan:** Klik "Simpan Nota". Data akan otomatis muncul di daftar bawah.
4.  **Cetak Struk:** Klik ikon **Printer** pada kartu nota, lalu klik "Cetak Struk".
5.  **Hapus:** Klik ikon `X` untuk menghapus catatan yang salah.
